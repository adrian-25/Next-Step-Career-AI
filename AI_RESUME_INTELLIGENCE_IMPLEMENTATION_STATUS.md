  - 4 color schemes: 'default', 'traffic', 'gradient', 'class'
  - Default: Green (80+), Yellow (60-79), Orange (40-59), Red (<40)
  - Traffic: Simple traffic light colors
  - Gradient: Smooth RGB gradient from red to green
  - Class: Returns Tailwind CSS class names
  - Example: `getScoreColor(85)` → "#22c55e" (green)

  - Suitable for cards and badges
  - Returns hex codes or Tailwind classes
  - Example: `getScoreBackgroundColor(85, true)` → "bg-green-100"

- `compareScores(currentScore, previousScore)` - Compares two scores
  - Returns: difference, percentChange, direction, improved, formattedDifference
  - Direction: 'up', 'down', or 'same'
  - Example: `compareScores(85, 75)` → `{ difference: 10, percentChange: 13.3, direction: 'up', improved: true }`

  - 80+: "Excellent"

  - 40-59: "Needs Improvement"
  - <40: "Poor"

  - Takes array of `{ value, weight }` objects

  - Example: `[{ value: 80, weight: 0.4 }, { value: 70, weight: 0.6 }]` → 74

- `getScoreTrend(scores)` - Analyzes historical scores for trends
  - Uses simple linear regression to calculate slope
  - Returns: trend ('improving', 'declining', 'stable'), slope, consistency
  - Consistency based on standard deviation: 'high' (<5), 'medium' (<10), 'low' (≥10)

- `normalizeScore(score, fromRange, toRange)` - Converts scores between scales
  - Example: `normalizeScore(75, [0, 100], [0, 10])` → 7.5
  - Clamps to input range before normalization
